== Description ==

### Apijoin Gumroad

Display your Gumroad products easily with Wordpress Gutenberg



### How to use

After you succesfully installed, you will see the new gumroad blocks under the apijoin gumroad category. Just insert any of the available blocks and put your product id inside the block options to display the proper product.



== Installation ==

Automatic installation is the easiest option as WordPress handles the file transfers itself and you do not need to leave your web browser. To do an automatic install of Apijoin Gutenberg, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
